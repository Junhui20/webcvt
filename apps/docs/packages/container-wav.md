# @webcvt/container-wav

> WAV (RIFF/WAVE) container parser and writer for webcvt.

## Installation

```bash
npm i @webcvt/container-wav
```

## API

Detailed API reference coming in v0.2. See the [source code](https://github.com/Junhui20/webcvt/tree/main/packages/container-wav/src) for now.

## Notes

Supports PCM (integer and float), WAVE_FORMAT_EXTENSIBLE, and multi-channel audio. Pure TypeScript — no WASM required for WAV encode/decode.
