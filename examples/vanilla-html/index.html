webcvt — SRT → VTT in the browser

A minimal example: no build tool, no bundler. One HTML file, ~15 lines of JavaScript.

<script type="module">
  import { parseSrt, serializeVtt } from 'https://esm.sh/@webcvt/subtitle@0.1.0';
  const srt = await file.text();
  const vtt = serializeVtt(parseSrt(srt));
</script>
